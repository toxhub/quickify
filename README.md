# QuickIFY
C++ based HTTP/3 Client

## Dependencies
- [libuv](https://github.com/libuv/libuv) - Cross-platform asynchronous I/O

## Build

### Fetch source code

```sh
$ git clone git@github.com:toxhub/quickify.git
$ cd quickify
```

### Build QuickIFY

```sh
$ ./configure
$ make
```

### Run

```sh
./quickify
```
